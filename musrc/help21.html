MU-21 — Variable Arithmetic

This program simplifies keyboard operations by making the user-defined keys A through E directly for storage and recall of variables. For example the simple key sequence n E' A defines key A as n. The variable n can then be recalled for use at any point in a calculation by pressing A.

To illustrate how the program works, assume a rectangle of length L and width W. The values of L and W can be entered using:

L E' A

W E' B

Then:

Adisplay L Bdisplay W A x Barea 2 x A + 2 x Bperimeter A x² + B x² = √xdiagonal

L E' A

Variables can be calculated using subroutines that you enter in program memory. For example after A and B have been entered, the area of the rectangle can be calculated as variable C using the key sequence A' C if the following subroutine is entered into program memory.

Lbl C RCL 21 x RCL 22 = STO 23 INV SBR

Note that variables A through E reside in data register R21 through R25 respectively.

  • To store variable A, press E' A. Likewise for all variables.
  • To recall variable A, press A. Likewise for all variables.
  • To compute variable A, press A' A. Likewise for all variables.

Registers used: 00-01
